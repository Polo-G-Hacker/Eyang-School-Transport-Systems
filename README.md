# Eyang School Transport System (ESTS)

Full-stack school transport management system for **Saint Jean Ingénieur** — monthly subscription, QR-code boarding, OpenStreetMap tracking, and offline-first driver scanning.

The full implementation (Express + PostgreSQL backend, Ionic + Angular mobile app, migrations, seed data, Leaflet map, offline sync) is delivered via the first pull request against this `main` branch.

See the open PR for the complete README, API reference, schema overview, and setup instructions.
